# Templates

个人项目脚手架合集,用于快速启动新项目,避免每次从零搭建重复的基础配置(依赖引入、全局异常处理、统一返回格式等)。

## 目录结构

```
templates/
└── spring-boot-basic/     # Java Spring Boot 后端模板
```

后续会陆续加入前端(Vue3)、移动端(Flutter)等模板。

## spring-boot-basic

一个开箱即用的 Spring Boot 后端脚手架。
