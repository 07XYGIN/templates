开箱即用的 FastAPI 后端脚手架

<div align="center">
  <img src="https://img.shields.io/badge/Python-3.14%2B-3776AB?style=flat-square&logo=python" alt="Python 3.14+"/>
  <img src="https://img.shields.io/badge/FastAPI-0.139%2B-009688?style=flat-square&logo=fastapi" alt="FastAPI 0.139+"/>

